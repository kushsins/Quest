# Quest

> Modern Support Ticket Management Platform

Quest is a modern ticket management application focused on clean UI, excellent user experience, and a scalable backend architecture.

## Tech Stack

### Frontend
- React
- Vite
- TypeScript
- React Router
- TanStack Query
- Axios

### Backend
- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication

### Infrastructure
- Docker
- Git
- Cursor
